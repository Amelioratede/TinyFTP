# TinyFTP
该开源项目是纯为兴趣而做的，模仿国外的LeapFTP软件写的，GPL-3.0开源协议
## 开发环境
- **Qt 4.8.5 32bit**
- **vs2010**
## 已实现
- **上传**
- **下载**
- **刷新**
- **删除**
- **新建文件夹**
- **重命名**
## 未实现
还有些细节部分仍在进行开发，但由于本着对项目负责的态度，想要把它做得完善一些，不想就此终结，希望能够找到志同道合之士，我们一块继续进行软件的完善工作
